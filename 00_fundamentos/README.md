# Fundamentos

Modelos de IA são, no fundo, equações matemáticas ajustadas a dados. Para entendê-los de verdade — não apenas usá-los — é preciso dominar a linguagem em que foram escritos: probabilidade, álgebra linear, cálculo e programação.

Este módulo cobre esses fundamentos na ordem em que se tornam necessários, conectando cada conceito ao modelo de IA que o utiliza.
