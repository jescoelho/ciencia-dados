# IA Generativa

IA generativa é o ramo da inteligência artificial que trata de criar — não classificar. Em vez de aprender a distinguir entre classes de exemplos existentes, um modelo generativo aprende a distribuição dos próprios dados e pode amostrar exemplos novos que nunca existiram, mas são estatisticamente consistentes com o que foi observado.

É o pilar que fundamenta modelos como GPT, DALL-E, Stable Diffusion e os sistemas de agentes autônomos. Em risco e tesouraria bancários, as aplicações incluem geração de cenários sintéticos para stress testing, aumento de datasets com classes raras, anonimização com preservação de correlações estatísticas e agentes que consultam bases de conhecimento proprietárias.
